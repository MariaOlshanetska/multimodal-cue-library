# Multimodal Cue Library

This repository contains an exploratory library of atomic non-linguistic
communicative cues designed for controlled multimodal experiments and
humanâ€“agent interaction.

The library is organized as:
- one YAML file per cue
- a shared schema describing required fields
- explicit annotation of intent, context, and predicted perceptual outcomes

The current version reflects a Catalan interactional baseline and is
intended to scale cross-culturally.
